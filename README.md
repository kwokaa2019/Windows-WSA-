【安裝方法】

1.開啟微軟商店 更新至最新版...

2.啟用 Hyper-V 虛擬譏平台功能 ， 方法如下

以滑鼠右鍵按一下 Windows 鍵，然後選取 \[應用程式與功能\]→選取相關設定下方右側的 [程式和功能 ]→選取 [開啟或關閉 Windows 功能]→選取 [Hyper-V]，然後按一下 [確定]→安裝完成時，系統會提示您重新啟動您的電腦。
